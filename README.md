# OS Scheduling Visualizer
This project implements and visualizes different CPU scheduling algorithms, including:

- First Come First Serve (FCFS)
- Shortest Job First (SJF)
- Shortest Remaining Job First (SRJF)
- Round Robin (RR)
- Priority Scheduling (Preemptive & Non-Preemptive)

## Features
- Interactive UI to select and input process data
- Generates a **Gantt Chart** and **Timeline Chart**
- Displays calculated metrics like turnaround time, waiting time, and CPU utilization

1. Clone the repository:
2. Open `index.html` in a browser.
